**1. What does `import random` do?**
A. Deletes all variables | B. Loads the `random` module for use | C. Starts a loop | D. Creates a string
<Answer: B. Loads the `random` module for use>

**2. What is a library in Python?**
A. A repeated block of code only for loops | B. A collection of reusable code | C. A file that can only store text | D. A type of exception
<Answer: B. A collection of reusable code>

**3. Where are command-line arguments commonly stored when using `sys`?**
A. sys.argv | B. sys.print | C. sys.len | D. sys.range
<Answer: A. sys.argv>

**4. Why might a programmer create their own library file?**
A. To avoid all functions | B. To organize reusable code in one place | C. To replace Python itself | D. To stop using imports
<Answer: B. To organize reusable code in one place>
