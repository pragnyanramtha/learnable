**1. What is the purpose of a unit test?**
A. To color-code the terminal | B. To check whether a small part of a program behaves correctly | C. To rename files automatically | D. To replace functions
<Answer: B. To check whether a small part of a program behaves correctly>

**2. What does `assert square(2) == 4` check?**
A. That `square(2)` returns `4` | B. That `square(2)` returns `2` | C. That `4` is a string | D. That the file is imported twice
<Answer: A. That `square(2)` returns `4`>
