**1. Which Python function is commonly used to display text on the screen?**
A. input | B. print | C. len | D. range
<Answer: B. print>
<Explanation: The print function is used to send output to the screen, while input asks the user for information.>

**2. In the line `name = input("What's your name? ")`, what does `name` represent?**
A. A comment | B. A variable storing the user's input | C. A built-in number type | D. A loop counter
<Answer: B. A variable storing the user's input>
<Explanation: The variable name stores whatever text the user enters so the program can use it later.>

**3. What is a string in Python?**
A. A repeated loop | B. A decimal number | C. Text stored inside quotation marks | D. A function definition
<Answer: C. Text stored inside quotation marks>
<Explanation: A string is Python's basic text type and is usually written inside single or double quotation marks.>

**4. Why are comments useful in code?**
A. They make Python run faster | B. They explain intent to the programmer and readers | C. They replace variables | D. They are required after every line
<Answer: B. They explain intent to the programmer and readers>
<Explanation: Comments are for humans, not for Python execution, and they help explain what the code is meant to do.>

**5. What does the equal sign do in `name = input(...)`?**
A. Compares two values | B. Ends the program | C. Assigns the value on the right to the name on the left | D. Starts a loop
<Answer: C. Assigns the value on the right to the name on the left>
<Explanation: In Python, a single equal sign performs assignment, meaning it stores a value in a variable.>

**6. Which function is used to ask the user for information?**
A. print | B. input | C. int | D. def
<Answer: B. input>
<Explanation: The input function displays a prompt and waits for the user to type a response.>

**7. What is the result of calling `input()`?**
A. A string returned from the user | B. A random number | C. A comment line | D. A loop object
<Answer: A. A string returned from the user>
<Explanation: Input returns text, so even numeric-looking input starts as a string until you convert it.>

**8. Why does `print("hello, name")` not show the user's actual name?**
A. Because `print` cannot show strings | B. Because `name` is inside quotation marks and is treated as text | C. Because variables must be uppercase | D. Because input was never called
<Answer: B. Because `name` is inside quotation marks and is treated as text>
<Explanation: Anything inside quotation marks is treated as literal text, not as a variable reference.>

**9. What is pseudocode mainly used for?**
A. Running Python faster | B. Replacing all comments | C. Planning steps in plain language before coding | D. Installing libraries
<Answer: C. Planning steps in plain language before coding>
<Explanation: Pseudocode helps you think through the logic before you worry about exact Python syntax.>

**10. Which choice best describes a function in Python?**
A. A stored comment | B. An action Python can perform | C. A type of loop only | D. A saved file
<Answer: B. An action Python can perform>
<Explanation: Functions are reusable actions, such as printing output or converting data from one type to another.>

**11. Which of these is an integer?**
A. "42" | B. 42 | C. "hello" | D. 3.14
<Answer: B. 42>
<Explanation: An integer is a whole number without quotation marks and without a decimal point.>

**12. Which of these is a float?**
A. 7 | B. "7.5" | C. 7.5 | D. input
<Answer: C. 7.5>
<Explanation: A float is a number that includes a decimal point.>

**13. Why is readable code important?**
A. It changes how Python calculates numbers | B. It makes code easier for people to understand and maintain | C. It removes the need for testing | D. It prevents all bugs
<Answer: B. It makes code easier for people to understand and maintain>
<Explanation: Readable code helps both the original programmer and future readers understand what the program is doing.>

**14. What does `def` do in Python?**
A. Deletes a variable | B. Defines a function | C. Declares a comment | D. Draws a diagram
<Answer: B. Defines a function>
<Explanation: The def keyword is used to create your own function so code can be reused more easily.>

**15. What is a good reason to use a variable?**
A. To make programs longer | B. To store a value so it can be reused later | C. To avoid all functions | D. To turn comments into output
<Answer: B. To store a value so it can be reused later>
<Explanation: Variables give a name to a value so the program can refer to it again without asking for or recalculating it.>
