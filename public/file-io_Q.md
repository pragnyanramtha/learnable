**1. What does File I/O mean?**
A. Reading from and writing to files | B. Importing only image libraries | C. Repeating a loop until it stops | D. Matching text patterns
<Answer: A. Reading from and writing to files>

**2. In `open("names.txt", "a")`, what does `"a"` mean?**
A. Analyze | B. Append | C. Average | D. Argument
<Answer: B. Append>

**3. Why is `with open(...)` often preferred?**
A. It automatically handles cleanup like closing the file | B. It removes all file contents | C. It changes strings into loops | D. It prevents reading files
<Answer: A. It automatically handles cleanup like closing the file>

**4. What kind of file is a CSV file?**
A. A file for only Python code | B. A binary image file | C. A comma-separated values file for structured data | D. A file that cannot be sorted
<Answer: C. A comma-separated values file for structured data>
