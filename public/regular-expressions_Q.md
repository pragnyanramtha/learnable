**1. What is a regular expression mainly used for?**
A. Building classes | B. Matching text patterns | C. Closing files | D. Creating global variables
<Answer: B. Matching text patterns>

**2. Which Python module is commonly used for regex?**
A. re | B. sys | C. math | D. csv
<Answer: A. re>

**3. In many regex patterns, what does `^` usually represent?**
A. End of string | B. Any decimal number | C. Start of string | D. A Python comment
<Answer: C. Start of string>

**4. Why should input often be cleaned before regex matching?**
A. Because regex cannot read strings | B. Because extra spaces or formatting can affect matches | C. Because regex only works on numbers | D. Because it removes the need for patterns
<Answer: B. Because extra spaces or formatting can affect matches>
