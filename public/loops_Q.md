**1. Which loop continues running while its condition stays true?**
A. if | B. while | C. else | D. match
<Answer: B. while>

**2. What is `range(3)` commonly used for in a `for` loop?**
A. Reading files | B. Creating comments | C. Repeating something three times | D. Catching exceptions
<Answer: C. Repeating something three times>

**3. What does `len()` tell you?**
A. Whether a value is valid | B. The number of items in a collection | C. The type of a variable | D. The remainder after division
<Answer: B. The number of items in a collection>

**4. What is a dictionary in Python?**
A. A collection of key-value pairs | B. A loop that never stops | C. A type of comment | D. A command-line argument
<Answer: A. A collection of key-value pairs>
