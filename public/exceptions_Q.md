**1. What is an exception in Python?**
A. A type of loop | B. A runtime error or problem during execution | C. A comment inside code | D. A package from the internet
<Answer: B. A runtime error or problem during execution>

**2. Which block contains the code that may fail?**
A. else | B. pass | C. try | D. import
<Answer: C. try>

**3. When does an `else` block after `try` run?**
A. Only when the `try` block succeeds | B. Only when an exception occurs | C. Before the `try` block | D. Never in Python
<Answer: A. Only when the `try` block succeeds>

**4. What is `pass` used for?**
A. To repeat a loop | B. To return a value | C. To do nothing as a placeholder | D. To print an error
<Answer: C. To do nothing as a placeholder>
