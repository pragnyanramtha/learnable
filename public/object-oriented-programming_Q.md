**1. What is a class in object-oriented programming?**
A. A blueprint for creating objects | B. A file-reading mode | C. A type of loop | D. A command-line argument
<Answer: A. A blueprint for creating objects>

**2. What does `self` usually refer to inside a class method?**
A. The Python interpreter | B. The current object instance | C. A global variable | D. The file name
<Answer: B. The current object instance>

**3. What is inheritance used for?**
A. Writing comments faster | B. Letting one class build on another | C. Avoiding all methods | D. Converting strings into integers
<Answer: B. Letting one class build on another>

**4. Why might a class raise an exception?**
A. To make code less readable | B. To reject invalid data or state | C. To repeat a loop | D. To import a package
<Answer: B. To reject invalid data or state>
