**1. What is a set useful for in Python?**
A. Storing duplicate values in order | B. Storing unique values | C. Replacing all functions | D. Handling image files only
<Answer: B. Storing unique values>

**2. What does a type hint do?**
A. It runs code automatically | B. It explains expected input or output types | C. It creates a file | D. It removes docstrings
<Answer: B. It explains expected input or output types>

**3. What does `*args` collect?**
A. Extra positional arguments | B. Only dictionary keys | C. Only file names | D. Exception messages
<Answer: A. Extra positional arguments>

**4. What is one benefit of a generator?**
A. It always returns all values at once | B. It can produce values one at a time and save memory | C. It replaces every loop with recursion | D. It deletes duplicates automatically
<Answer: B. It can produce values one at a time and save memory>
