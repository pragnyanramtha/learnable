**1. Which keyword starts a decision block in Python?**
A. for | B. while | C. if | D. with
<Answer: C. if>
<Explanation: The if keyword begins a conditional block and tells Python to check whether a condition is true.>

**2. What does `elif` do?**
A. It repeats code forever | B. It checks another condition if an earlier one was false | C. It ends the program | D. It creates a variable
<Answer: B. It checks another condition if an earlier one was false>
<Explanation: Elif gives Python another condition to test when the earlier if condition did not match.>

**3. What does the modulo operator `%` return?**
A. A random number | B. The product of two numbers | C. The remainder after division | D. A string comparison result
<Answer: C. The remainder after division>
<Explanation: Modulo gives what is left over after one number is divided by another, which is useful for parity checks.>

**4. In a conditional expression, what does `and` mean?**
A. Both conditions must be true | B. At least one condition must be true | C. The condition must be false | D. The result is a loop
<Answer: A. Both conditions must be true>
<Explanation: And combines tests so every connected condition must be true for the whole expression to be true.>

**5. What does `else` do in a conditional chain?**
A. Runs only if all earlier conditions were false | B. Repeats code forever | C. Imports a module | D. Converts a string to an integer
<Answer: A. Runs only if all earlier conditions were false>
<Explanation: The else block acts as the fallback path when no previous if or elif condition matched.>

**6. Which operator checks whether two values are equal?**
A. = | B. == | C. != | D. >=
<Answer: B. ==>
<Explanation: Double equals checks equality, while a single equals sign performs assignment.>

**7. What does `or` mean in a condition?**
A. Both conditions must be false | B. Both conditions must be true | C. At least one condition must be true | D. The condition is skipped
<Answer: C. At least one condition must be true>
<Explanation: Or allows the whole expression to pass if any one of the linked conditions is true.>

**8. Which of these conditions checks whether `x` is not equal to `5`?**
A. x = 5 | B. x == 5 | C. x != 5 | D. x <= 5
<Answer: C. x != 5>
<Explanation: The exclamation mark plus equals sign means not equal to.>

**9. Why is `% 2 == 0` often used?**
A. To check whether a number is even | B. To create a string | C. To open a file | D. To sort a list
<Answer: A. To check whether a number is even>
<Explanation: If a number leaves remainder zero when divided by two, then it is even.>

**10. What is control flow?**
A. The order in which a program chooses and runs steps | B. A library for graphics | C. A type of comment | D. A file extension
<Answer: A. The order in which a program chooses and runs steps>
<Explanation: Control flow describes how execution moves through different branches based on conditions.>

**11. What happens if the condition in an `if` statement is false and there is no `else`?**
A. Python stops permanently | B. Python skips the indented block and continues | C. Python restarts the file | D. Python creates a variable automatically
<Answer: B. Python skips the indented block and continues>
<Explanation: If the condition is false, Python simply moves on unless there is another branch to check.>

**12. Which comparison checks whether `x` is greater than or equal to `10`?**
A. x > 10 | B. x < 10 | C. x >= 10 | D. x == 10
<Answer: C. x >= 10>
<Explanation: Greater than or equal to uses the combined operator greater-than followed by equals.>

**13. Why might a programmer place a condition inside a custom function like `is_even`?**
A. To make the code reusable and clearer | B. To prevent all exceptions | C. To avoid using return values | D. To remove indentation
<Answer: A. To make the code reusable and clearer>
<Explanation: A custom function can hide repeated logic behind a descriptive name and make the main program easier to read.>

**14. What does indentation mean after an `if` line?**
A. The program should ignore the next line | B. The indented lines belong to that conditional block | C. The file is invalid | D. The code is a comment
<Answer: B. The indented lines belong to that conditional block>
<Explanation: Indentation defines which lines run as part of the if, elif, or else block.>

**15. When is `match` useful?**
A. When comparing one value against several known cases | B. Only for loops | C. Only for file input | D. Only for comments
<Answer: A. When comparing one value against several known cases>
<Explanation: Match can make multi-way decision logic more readable when one variable is being checked against several possibilities.>
